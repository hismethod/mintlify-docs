## Base URL

인증 API는 HTTPS를 통해 제공됩니다.

<Note>문서의모든 URL의 기본은 https://api.furo.one 입니다.</Note>

## Authentication

인증 API를 사용하면 Furo를 사용할 때 사용자 ID의 모든 측면을 관리할 수 있습니다. 사용자가 로그인, 가입, API 액세스 등을 할 수 있도록 엔드포인트를 제공합니다.
API의 인가는 기본적으로 **Basic Auth** 방식으로 합니다. **JWT Bearer** 방식의 인증을 사용하는 API도 있습니다. 이런 API들에는 HTTP header 별도의 안내가 적혀있습니다.

## API Key

| Key             | 사용하는 API   | 공개 가능 여부 |
| --------------- | -------------- | -------------- |
| Public API Key  | Auth API       | O              |
| Private API Key | Management API | X              |

Client ID와 Public API Key/Admin API Key는 Furo Console의 개발자 도구 탭에서 확인하실 수 있습니다.
