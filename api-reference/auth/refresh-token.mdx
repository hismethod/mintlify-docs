---
title: 'Refresh Token'
api: 'POST /sessions/token/refresh'
description: 'Refreshes the token for the current user.'
authMethod: 'bearer'
---

### Header

<ParamField header="Authorization" type="string" required>
  This is the refresh token for the current user.
</ParamField>

### Body

<ParamField body="accessToken" type="string" required>
  This is the access token for the current user.
</ParamField>
