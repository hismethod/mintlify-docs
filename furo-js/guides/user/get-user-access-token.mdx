---
title: 'Get User Access Token'
description: '사용자의 액세스 토큰을 얻는 방법'
---

사용자를 인증하고 서버와 통신하기 위해서는 사용자의 액세스 토큰이 필요합니다.

사용자의 액세스 토큰을 얻기 위해서는 `furo` 객체에 있는 `getAccessTokenSilently` 메서드를 호출하여 얻을 수 있습니다.

```js
const jwtAccessToken = await furo.getAccessTokenSilently();
```

사용자가 로그인 중이라면 항상 유효한 토큰이 반환됩니다.
만약 사용자가 로그인 중이 아니라면 `null`이 반환됩니다.

사용자의 토큰을 검증하는 방법과 토큰에 대한 더 자세한 내용은 [여기](/essentials/token-verify)를 참고하세요.

<Accordion title="항상 유효한 토큰을 반환하는 이유">
  이 메서드는 내부적으로 `refresh_token`을 활용하여 토큰을 갱신합니다. 따라서
  저장된 토큰이 만료되었다면 자동으로 토큰의 유효기간을 갱신한 후 반환합니다.
</Accordion>
