Slack 연동을 통해서 Furo 프로젝트에 새로운 사용자가 가입 할때마다 채널에 메시지를 받아 보실 수 있습니다. Slack에 앱을 추가하여 간편하게 연동해보세요.

## 1. Slack 앱 추가

연동할 Slack 워크스페이스에서 앱을 추가합니다.

<img src="/images/slack1.png" alt="Slack" />

검색창에 Incoming WebHooks를 검색한 후 추가 버튼을 누릅니다.

<img src="/images/slack2.png" alt="Slack" />

Slack에 추가 버튼을 누릅니다.

<img src="/images/slack3.webp" alt="Slack" />

연동할 채널을 선택 혹은 새 채널 생성을 눌러 채널을 생성한 후 수신 웹후크 통합 앱 추가 버튼을 누릅니다.

<img src="/images/slack4.png" alt="Slack" />

웹후크 URL 항목에서 URL을 복사합니다.

<img src="/images/slack5.png" alt="Slack" />

복사한 URL을 Furo Console 설정 탭의 Slack 연동 API Key 입력란에 입력하고 Save 버튼을 눌러 저장합니다.

<img src="/images/slack6.jpeg" alt="Slack" />

Furo를 통해 회원가입을 할 경우 해당 채널에서 회원가입 알림을 받으실 수 있습니다.

<img src="/images/slack7.webp" alt="Slack" />
