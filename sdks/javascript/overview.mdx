Furo Javascript SDK는 웹 애플리케이션에서 쉽게 IAM(Identity and Access Management) 기능을 통합할 수 있도록 설계되었습니다. 이 SDK는 OAuth 로그인, 사용자 관리, 토큰 관리 등을 포함한 다양한 기능을 제공합니다.

## 설치

<Warning>[Furo Console](https://console.furo.one)의 홈화면에서 Javascript SDK 아이콘을 선택하시면 여러분의 프로젝트에 맞게 설정된 Script를 복사하여 그대로 사용하실 수 있습니다.</Warning>
Javascript SDK는 스크립트 태그를 사용하여 웹사이트에 로드할 수 있습니다. 사이트 `<head>` 요소에 다음 Script를 추가하세요.

```html
<script src="https://cdn.furo.one/furo-js-sdk/furo.min.js" data-client-id="{YOUR_CLIENT_ID}" data-public-api-key="{YOUR_PUBLIC_API_KEY}" />
```

## 사용법

Furo Javascript SDK는 `Furo` 전역 객체를 통해 사용할 수 있습니다. `Furo` 객체는 다양한 속성과 메소드를 제공합니다. 아래 내용을 참고하여 SDK를 사용하세요.

---

## 속성 (Properties)

| 속성 | 데이터 타입 | 설명 |
| --- | --- | --- |
| `version` | `string` | SDK의 현재 버전을 나타냅니다. |
| `isLoaded` | `boolean` | SDK가 성공적으로 로드되었는지 여부를 나타냅니다. 초기화가 완료되면 `true`를 반환합니다. |
| `user` | `User` \| `null` | 현재 로그인한 사용자에 대한 정보를 담은 [User](/api-reference/data/user) 객체입니다. 사용자가 로그인하지 않은 경우 `null`을 반환합니다. |
| `workspace` | `Workspace` \| `null` | 현재 워크스페이스에 대한 정보를 담은 [Workspace](/api-reference/data/workspace) 객체입니다. 워크스페이스가 설정되지 않은 경우 `null`을 반환합니다. |
| `isAuthenticated` | `boolean` | 로그인 한 경우 `true`, 로그인하지 않은 경우 `false`를 반환합니다. |

---

## 메소드 (Methods)

### 일반 메소드

#### `isReady()`

SDK가 초기화되고 사용 준비가 되었는지 여부를 반환합니다.

**반환값**: `true` 또는 `false`

```javascript
if (window.Furo.isReady()) {
  // SDK 사용 준비 완료
}
```

#### `init(options)`

SDK를 초기화합니다. 설정 옵션을 파라메터로 받습니다.

**파라메터**: `{ clientId: String, publicKey: String }`

```javascript
window.Furo.init({
  clientId: "{YOUR_CLIENT_ID}",
  publicKey: "{YOUR_PUBLIC_API_KEY}",
});
```

#### `logout()`

현재 사용자를 로그아웃시킵니다.

```javascript
window.Furo.logout();
```

### 로그인 UI 컴포넌트

<Warning>여기에 로그인 UI 이미지</Warning>

#### `mountLogin(containerId)`

지정된 컨테이너에 로그인 UI를 마운트합니다.

**파라메터**: `containerId` (로그인 UI를 마운트할 HTML 요소의 id)

```html
<script>
  const loginContainer = document.querySelector("#login-container");
  window.Furo.mountLogin(loginContainer);
</script>
```

#### `openLogin()`

로그인 UI를 화면에 표시합니다.

```javascript
window.Furo.openLogin();
```

#### `closeLogin()`

열려있는 로그인 UI를 화면에서 보이지 않게합니다.

```javascript
window.Furo.closeLogin();
```
